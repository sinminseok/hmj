## Find Life 프로젝트

-----

### 프로젝트 설명
`사용자들이 호주에서 집과 일자리를 쉽게 구할 수 있게 도움을 주는 서비스`

한인들을 대상으로 서비스할 예정이며 수요가 확인되면 호주 현지인들에게도 제공할 예정.

### 프로젝트 설정

```
- Spring Version : 3.2.2
- Java Version : 17
- DataBase : (MySql) 8.0.27

- Server : Aws Ec2
```

### 프로젝트 구조
계층형 구조이면 프로젝트가 커질 수록 클래스들이 응집될 수 있기에 `도메인형 구조` 선택
